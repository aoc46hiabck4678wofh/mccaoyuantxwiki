# 二、作弊行为分类
## 一般类作弊行为
- **非法入侵**: 以非法手段进入他人私人领地
- **盗窃**: 以非法手段进入他人领地并盗窃财物且未造成严重后果
- **扰乱交易市场** ：扰乱交易市场，影响他人正常交易

## 严重类作弊行为
- **杀戮光环**：使用非法客户端、插件、mod等相关工具，影响他人正常游戏
- **违规飞行**：使用非法客户端、插件、mod等相关工具，在服务器内飞行
- **其它近战类作弊行为**：含范围攻击、瞬移、瞬杀、透视等


# 三、作弊行为的处理
- **一般类作弊**：对于未造成严重后果的，给与其7天以上30天以下的封禁处理<br>对于造成严重后果的，给予其1年以上，3年以下封禁，并处没收全部财产
- **严重类作弊**：对于未造成严重后果的，给予其3年以上，5年以下封禁，可并处没收全部财产。<br>对于已造成严重后果的，给予其10年以上封禁并处没收全部财产
- **自首、重大立功表现**：对于作弊者存在的自首、重大立功表现，可依法从轻或减轻处罚。如若减轻处罚仍过重时，可依法免除处罚。
